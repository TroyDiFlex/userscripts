# 🛍️ Магазин скриптов Tampermonkey

Коллекция пользовательских скриптов для Tampermonkey.

## 🔗 Открыть магазин

👉 **[troydiflex.github.io/userscripts](https://troydiflex.github.io/userscripts)**

---
## ⚡ Установка

1. Установите [Tampermonkey](https://www.tampermonkey.net/) в браузер
2. Откройте [магазин](https://troydiflex.github.io/userscripts)
3. Нажмите **⬇️ Установить** на нужном скрипте
4. Подтвердите установку в Tampermonkey
