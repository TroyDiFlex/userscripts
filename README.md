# 🛍️ Магазин скриптов Tampermonkey

Коллекция пользовательских скриптов для Tampermonkey — помощники для Avito, TecAlliance и других сервисов.

## 🔗 Открыть магазин

👉 **[troydiflex.github.io/userscripts](https://troydiflex.github.io/userscripts)**

---

## 📜 Скрипты

| Скрипт | Описание |
|--------|----------|
| 🏷️ Помощник актуализации цен | Анализирует конкурентов на Avito и считает рекомендуемую цену |
| 📊 Avito Pro Tools | Подсвечивает карточки без продвижения, показывает стоимость просмотра |
| 📈 Avito Wordstat Автопарсер | Собирает статистику спроса по артикулам с Авито Wordstat |
| 🔢 TecAlliance — Скопировать номера | Копирует артикулы одной кнопкой |
| 📋 TecAlliance — Скопировать применимость | Копирует данные применимости в буфер |

---

## ⚡ Установка

1. Установите [Tampermonkey](https://www.tampermonkey.net/) в браузер
2. Откройте [магазин](https://troydiflex.github.io/userscripts)
3. Нажмите **⬇️ Установить** на нужном скрипте
4. Подтвердите установку в Tampermonkey
